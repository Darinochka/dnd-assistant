# DND ассистент

ИИ-ассистент, который помогает мастеру и игрокам вести кампанию в Dungeons & Dragons. Он понимает текстовые описания, генерирует сцены, отслеживает сюжетные ветки и предлагает события, адаптируясь к решениям игроков. Ассистент умеет быстро подбирать подходящих NPC, подсказывать правила, автоматически считать броски и формировать подсказки для импровизации, чтобы упростить подготовку и сделать сессии динамичными.
